Cmder安装

安装

Cmder = Cmd + PowerShell + Git for Windows + “bash”

cmder更加方便、可视的进行cmd等命令操作。

通过官网Cmder 下载cmder压缩包，解压即可。注意解压路径不能位于需要管理员访问权限的地方。

  • 添加环境变量：

    新建环境变量cmder_root，将其设置为cmder.exe，再讲%cmder_root%添加到系统的PATH环境变量中。

  • 讲cmder添加到文件夹右键菜单：

    用管理员权限打开cmd；进入到cmder的解压路径；.\cmder.exe /REGISTER ALL，即可添加。
